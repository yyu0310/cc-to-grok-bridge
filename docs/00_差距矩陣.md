## 一句話

Grok Build **原生就吃** 大部分 CC 資產；本橋補 **hook payload／deny**、**memory 單向鏡像**、**MCP 遷移手冊**，並用 doctor 守門。

## 九大域＋MCP 對照

| 域 | CC 現況 | Grok 現況 | 差距 | 橋接動作 |
|---|---|---|---|---|
| System Prompt | vault 根 `CLAUDE.md` | 自動載入（compat.claude） | 幾乎零 | 從 workspace 根啟動；Grok 專用 `.grok/rules/` |
| harness 制度 | 你的 harness 制度檔 | 靠模型讀檔 | 可攜 | 不動 |
| memory | `~/.claude/projects/<hash>/memory/` 開場載索引 | 鏡像 + **rules 指標**（開場必載）；產品 memory 搜尋可選 | **高**（日用好處理） | **CC→Grok pull** + 三區隔離 + 可選 `memory_push`；見 `docs/02_memory.md` |
| hook | settings.json + `~/.claude/hooks/*.sh` | bridge 包 adapter | **小**（adapter 後） | `hook_adapter.py` + `install_bridge.py`；`compat.claude.hooks=false` 防雙跑。**硬擋實測 PASS**（clasprc/.env） |
| skill | SKILL/ → `~/.claude/commands` | 掃 commands | 幾乎零 | doctor 驗 symlink；2 檔缺 frontmatter 可接受 |
| **plugins** | `enabledPlugins` + marketplace + 常 SessionStart always-on | 獨立 `grok plugin *`；預設不跟 CC 走 | **低** | **不自動同步**；**A** 原生 Grok plugin／**B** `.grok/rules` always-on（不推 slash-only）。見 `docs/06_plugins.md` |
| subagent | Task 等 | `spawn_subagent` | 中 | 方法論可攜；語法不同 |
| launchd | OS 層 | 不綁 | 零 | 不動 |
| 設定檔 | settings 白名單 | config.toml + folder-trust | 中 | 日用已設 memory/hooks |
| 多模型協作 | AI 應用方案 | 同 | 零 | 參照 |
| **MCP** | `~/.claude.json` + claude.ai 雲端 | `[mcp_servers.*]` | **中（AI 代裝）** | 型態分類見 `docs/03_mcp.md`／`AGENTS.md`；claude.ai 雲端不可攜；key 永不進腳本 |
