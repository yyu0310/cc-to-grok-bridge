# 01_絲滑啟動

## 每次開工

```bash
cd ~/path/to/your-workspace
grok
```

## 第一次／更新後

```bash
cd /path/to/cc-to-grok-bridge
python3 scripts/install_bridge.py
python3 scripts/memory_sync.py --workspace ~/path/to/your-workspace
python3 scripts/bridge_doctor.py --workspace ~/path/to/your-workspace
```

## 週期性

| 頻率 | 動作 |
|---|---|
| CC memory 大改後 | `python3 scripts/memory_sync.py` |
| 新增 CC hook 後 | `python3 scripts/install_bridge.py` 重包 |
| 懷疑壞掉 | `python3 scripts/bridge_doctor.py` |
